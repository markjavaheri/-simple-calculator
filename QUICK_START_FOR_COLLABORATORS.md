# 🚀 Quick Start for Collaborators

Welcome! Here's everything you need to start collaborating on the Simple Calculator project.

## 📧 What You'll Receive from the Project Owner

The project owner will share:
1. **GitHub Repository URL** (e.g., `https://github.com/username/simple-calculator`)
2. **Invitation email** to collaborate on the repository

## ✅ Step-by-Step Setup (5 minutes)

### Step 1: Accept the GitHub Invitation

1. Check your email for an invitation from GitHub
2. Click "Accept invitation"
3. You now have access to the repository!

### Step 2: Install Required Software

**You need:**
- [VS Code](https://code.visualstudio.com/) - Code editor
- [Git](https://git-scm.com/downloads) - Version control
- Bob extension for VS Code

**Installing Bob:**
1. Open VS Code
2. Click Extensions icon (left sidebar)
3. Search for "Bob"
4. Click "Install"

### Step 3: Clone the Repository

**Option A: Using VS Code (Easiest)**
1. Open VS Code
2. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
3. Type "Git: Clone"
4. Paste the repository URL: `https://github.com/username/simple-calculator`
5. Choose where to save it
6. Click "Open" when done

**Option B: Using Terminal**
```bash
git clone https://github.com/username/simple-calculator.git
cd simple-calculator
code .
```

### Step 4: Test the Calculator

1. In VS Code, right-click `index.html`
2. Select "Open with Live Server" (or just open in browser)
3. Try adding and subtracting numbers
4. It works? Great! You're ready to contribute!

## 🤖 Using Bob to Help You

Bob is your AI coding assistant. Activate it:
- Press `Ctrl+L` (Windows/Linux) or `Cmd+L` (Mac)

**Try asking Bob:**
```
Bob, explain what this calculator does
```

```
Bob, help me add a multiply button
```

## 🆘 Need Help?

**Stuck? Try these:**
1. **Ask Bob:** "Bob, I'm stuck on [problem]"
2. **Check the guides:** All documentation is in the repository
3. **Ask in Issues:** Post your question on GitHub
4. **Contact the team:** Comment on your assigned issue
