# Simple Calculator Project 🧮

A beginner-friendly calculator project for learning addition and subtraction with easy collaboration using Bob and GitHub.

## What This Project Does

This is a simple web-based calculator that can:
- ➕ Add two numbers
- ➖ Subtract two numbers
- Display results in a clean, easy-to-use interface

## Quick Start

1. Open `index.html` in your web browser
2. Enter two numbers
3. Click "Add" or "Subtract"
4. See your result!
